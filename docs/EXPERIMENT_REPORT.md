# Ananke — 实验报告（MVP v0.1）

> 版本：MVP v0.1（仪器发布版本） · 基于协议 [`01_PROTOCOL_v3.md`](01_PROTOCOL_v3.md) · git tag `v0.1`
> 本文档是求职叙事材料，所有数字指向**不可变 tag `v0.1`**。它刻意只声明能被证据支撑的结论。
> 经 GLM 系统性证伪审查后，凡超出证据支撑的措辞均已降级（见 §④）。

## ① 一句话命题

Ananke 是**约束场理论的实验仪器**：它验证的不是"记忆系统好不好用"，而是——当把"长期稳定结构依赖不可控外部输入的持续检验（External Selection）而非系统内部自循环（Internal Selection）"操作化为两套迁移规则时，**同一系统、同一输入流能否产生可判别的记忆结构动力学，且分歧方向与理论预测一致**。

## ② 对照实验设计

- **唯一变量 = Migration Rule（选择压力来源）**：`persistence`（External Selection，默认）/ `frequency`（Internal Selection，控制组）。两组共用同一 LLM、同一嵌入、同一语料、同一提取 / 检索 / 重组逻辑；frequency 仅切换"升层判据"为 `total_activation`（每次语义命中 ≥0.60 即 +1，不区分来源）。
- **同语料同 LLM，两次独立运行**：用同一份 `corpus_phase1.txt`（26 句英文，badminton×6 / Mochi×6+矛盾×2 / 噪声），仅切 `--strategy` 跑两遍。两遍由 Gemini 独立抽取，temperature=0。
- **提取集一致性（比较器有效性）**：两遍提取的记忆集**完全一致**（各 21 条，交集 21，无独有偶）。这排除了"两次运行抽到不同记忆"对升层差异的污染——升层分歧只能来自策略切换，而非提取非确定性。（注：GLM 称"提取一致是必然的"，不准确——两次独立 Gemini 调用、temperature=0 不保证位级确定性，21=21 是经验验证的事实；该条 GLM 与其另一条"temperature=0 不保证确定性"自相矛盾。）
- **控制变量**：LLM 同源、Temperature=0.0、Prompt 冻结、嵌入模型 `all-MiniLM-L6-v2`。

## ③ 对比表（核心结果，数字不可变）

| 指标 | persistence（External Selection） | frequency（Internal Selection） |
|---|---|---|
| 升巩固层记忆数 | **1** | **4** |
| 其中带外部验证（EV>0） | 1 / 1 | **1 / 4** |
| 纯内部自循环升层（EV=0） | 0 | **3** |
| 升层记忆示例 | "enjoys playing badminton on weekends"（EV=2, persist=3.10） | 上述 badminton + "User loves badminton"(EV=0, persist=1.10) / "adopted a cat named Mochi"(EV=0, persist=1.84) / "Mochi is playful and energetic"(EV=0, persist=1.47) |

**读表**：External Selection 升层的记忆全部带外部验证（受环境约束）；Internal Selection 把**零外部验证、存续完全由自身检索频率决定**的记忆也推上巩固层——正是理论预测的"退化为内部自循环"。分歧精确落在"纯内部激活记忆是否该升层"这一刀上。

## ④ 诚实边界（本实验证明了什么、没证明什么）

**证明了**：两套不等价的迁移规则在同一输入流上产生可判别的结构差异，且分歧方向与约束场理论预测一致。这一条已经值得写进材料——它是机制差异的实证，不是空泛主张。

**没有证明（须如实声明）**：
- **"哪种结构更稳定"未被测量**。协议 §5 真正的比较器是"存活记忆 vs 演化中外部世界后期发展的一致性"，本实验只测了"谁被升层"，从未做扰动或存活率检验。因此本文档称"Phase 3 先导实验（pilot）"，**不称"Phase 3 证据"**。GLM 审查中最有价值的一击即在此。
- **阈值语义耦合未分离**。persistence 阈值 3.0（纯 IA 记忆需约 8 次激活）与 frequency 阈值 3（仅需 3 次）相差约 3 倍，score 函数与阈值**共同变化**。单点数据无法分离"选择压力来源"与"阈值数值"各自的贡献。但需说明：frequency 策略的语义本就是"不区分来源"，若让它对 IA 要求 8 次，它就不再是 frequency 而是 persistence 变体——阈值差异是策略定义的一部分，非实验疏漏。稳健性留待阈值 sweep。
- **语料构造偏误**。语料由知情实验者手工构造（badminton 近义复述、Mochi 多样特质），升层结果形状被"措辞冗余度"预先决定，是确认偏误的结构性来源。MVP 阶段可接受（目的是验证机制能否产生差异），但必须声明——且这正是下一步接入外部语料（LoCoMo）的动机。
- **n=1、单次运行、零统计推断**；temperature=0 不保证 LLM 位级确定性。
- **两个表征空间**：EV 比较"原始输入 vs 提取记忆"，dedup 比较"提取物 vs 提取物"，同一 0.80 在两个不同相似度分布上使用（技术细节，MVP 不修）。

**反身性红线守住了**：实验只喂外部语料，系统生成内容永不计入 external_validation；本次无违反。

## ⑤ Future Work（接 GLM 审查）

- **阈值 sweep**：在 persistence/frequency 阈值组合网格上重跑，检验结论稳健性，分离"来源加权"与"阈值数值"贡献。
- **外部语料（LoCoMo）**：用公开对话基准替代手工构造语料，消除构造偏误，测自然分布下的表现。
- **稳定性 / 扰动检验（协议 §5 真比较器）**：设计"世界演化"语料，比较存活记忆 vs 后期外部发展的一致性，测量"稳定 vs 易被破坏"标签——这是完整 Phase 3 的任务。
- **Memory Identity 显式化（v0.2）**：把 0.80 同一性标准从隐含提升为协议一级概念，统一 EV / dedup / Memory Unit 边界（GPT 与 GLM 两次独立评审收敛到同一对象）。
- **Phase 2 Reply 闭环**：检索影响输出，闭合"慢层塑形快层"的最后一公里。

## 附：命题与理论的层级关系

约束场理论自限为"只预测可能性空间的形状"（分层 / 外部门控 / 单向阀），不预测具体记忆条目的内容。本实验在"形状"层给出证据；GLM 审查指出的"因果方向无法确立"（单次运行无法排除"语料结构决定哪些记忆存在、策略只决定去哪层"的反向解释）是 MVP 已知边界，由上述 Future Work（外部语料 + 稳定性检验）回应。

证伪审查的正确用法不是推翻结论，而是替结论修剪到刚好能防御的尺寸——GLM 的 17 条审查最终把结论收窄为："机制差异确实被证明（分歧方向与理论预测一致），但'哪种结构更稳定'未被测量。"
